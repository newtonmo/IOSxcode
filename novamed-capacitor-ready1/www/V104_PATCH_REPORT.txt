NovaMed V104 patch report

Source used:
- novamed-free-courses-v103-home-review-clean-ui.zip

Applied fixes:
1) Videos Free Courses button
- Removed the shared quick-exam-trigger class from the Videos Free Courses button.
- The Videos button now remains Free Courses and opens the Free Courses library only.
- Exam live status now targets only the QBank exam trigger (#quickExamTrigger), so it cannot overwrite the Videos Free Courses button.

2) Live exam resume/cancel controls
- Added live exam banners on Home, Videos, Learn, and QBank after starting an exam.
- Banner includes two actions:
  - Resume exam / Review result
  - Cancel exam
- Cancel exam clears the live exam without saving a score report.

3) Exam scroll fix
- Added a dedicated real-exam-modal-card modal variant.
- Timed exams now use a scroll-safe modal layout, especially on mobile.

4) QBank guided MCQ mode
- Added MCQ mode to the QBank guided path:
  - Exam style: answers stay hidden until the score report after all questions.
  - Free questions: opens the normal QBank tutor mode with instant feedback.
- The setting is saved in UI progress/student progress as selectedQAnswerMode.

5) Video guided lecture behavior
- Non-admin users can select a lecture from the Videos guided path and it opens directly.
- Admin still keeps the lecture list/admin editing workflow.

Verification:
- node --check passed for every JavaScript file.
- No Arabic text remains inside HTML/JS/CSS interface files.
- Cache/version strings were updated to V104.

NovaMed V105 patch report

Base version:
- novamed-free-courses-v104-exam-qbank-freecourses-fix.zip

Fixes applied:
1. QBank no longer auto-opens MCQs after choosing a lecture/subtopic card.
   - Selecting a lecture now only selects the path and shows the selected set card.
   - MCQs open only after pressing Open set.

2. QBank no longer auto-opens MCQs after page refresh.
   - Even if the saved state contains a selected lecture, renderQbankSet now shows the selected set card instead of opening the question player/modal automatically.

3. Open set now respects the chosen MCQ mode.
   - Exam style starts the exam-style flow with hidden answers until the report.
   - Free questions opens the existing QBank tutor/free-question player.

4. Next question click reliability improved.
   - Removed duplicate local Next question listeners and kept a single delegated handler.
   - Added CSS to prevent pseudo-elements/layers from stealing taps and to make the whole button area clickable.

5. Free Courses button design updated.
   - The button keeps the Free Courses function only.
   - Its visual style now matches the former Exam mode CTA style in the Videos screen.

Validation:
- node --check passed for all JavaScript files.
- No Arabic text found inside HTML/JS/CSS interface files.
- Service worker cache version updated to V105.
